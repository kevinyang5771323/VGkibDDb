# 前言

欢迎来到基于SSM的房屋租赁系统项目。本项目旨在为用户提供一个便捷、高效的房屋租赁在线平台。在这里，用户可以轻松发布房源信息，查找合适的租赁房源，实现房东与租客之间的无缝对接。

# 内容介绍

基于SSM的房屋租赁系统主要包括以下几个模块：用户模块、房源模块、租赁模块、支付模块等。系统采用前后端分离的开发模式，后端采用Java语言，Spring、SpringMVC和MyBatis框架，前端采用JS、Vue和CSS3技术。以下是各模块的简要介绍：

1. 用户模块：提供用户注册、登录、个人信息管理等功能。
2. 房源模块：提供房源发布、编辑、删除等功能，以及房源搜索、筛选等功能。
3. 租赁模块：实现房东与租客之间的租赁关系，包括租赁合同签订、租金支付等。
4. 支付模块：对接第三方支付平台，实现租金的在线支付。

# 技术介绍

- 语言：Java
- 使用框架：Spring、SpringMVC、MyBatis
- 前端技术：JS、Vue、CSS3
- 开发工具：IDEA/Eclipse
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12/14/16
